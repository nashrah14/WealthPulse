# 📌 Project Overview

**WealthPulse** is a clean, modern financial dashboard designed to simulate real-world fintech applications.  
This project focuses on **frontend architecture, UI/UX, responsiveness, and state management**.

The dashboard enables users to:

- Track financial summary
- Analyze spending patterns
- View insights
- Manage transactions
- Switch roles (Admin / Viewer)
- Experience responsive fintech UI
