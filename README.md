# 📌 Overview

**WealthPulse** is a clean and interactive finance dashboard that helps users track their financial activity, analyze spending patterns, and manage financial goals.

This project focuses on **frontend architecture, UI/UX design, and state management**, while simulating real-world dashboard functionality.

The goal was to build a **modern, responsive, and intuitive financial dashboard** that demonstrates strong frontend development skills and thoughtful user experience.

---

# ✨ Key Features

## 📊 Dashboard Overview
- Total Balance
- Total Income
- Total Expenses
- Savings Rate
- Financial Health Indicator

## 📈 Analytics & Visualizations
- Income vs Expense Trend (Line Chart)
- Spending by Category (Pie Chart)
- Animated and responsive charts
- Smooth transitions and hover animations

## 💳 Transaction Management
- Add transactions
- Track income & expenses
- Delete transactions
- Search transactions
- Filter transactions
- Sort transactions
- Pagination support

## 🎯 Goal Tracking
- Create financial goals
- Set target amount
- Set deadline
- View goals in dashboard
- Persistent goals using local storage

## 👥 Role-Based UI
- Admin Role
  - Add transactions
  - Create goals
  - Edit financial data

- Viewer Role
  - View-only access
  - No modification permissions

Role switching helps simulate real-world dashboard behavior.

---

# 🧠 Technical Decisions

### Why React + TypeScript
- Strong component architecture
- Better scalability
- Type safety
- Cleaner codebase

### Why Tailwind CSS
- Fast UI development
- Consistent design system
- Responsive-first approach

### Why Zustand
- Lightweight state management
- Minimal boilerplate
- Easy global state handling

### Why Local Persistence
- Simulates real dashboard behavior
- No backend dependency
- Keeps project frontend-focused
